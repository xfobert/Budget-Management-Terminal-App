# Budget-Management-Terminal-App

Budget is a Python command-line budget management tool designed to keep track of your monthly expenses, income and savings.
Data is stored in JSON format and can be exported to CSV for use in Excel or Google Sheets.

## Features:

Create, update, and delete monthly budget records
View all saved records sorted by month
Calculate savings automatically
Display income, expense, and savings statistics
View monthly spending trends
Export data to CSV
Persistent JSON storage

## Built with

Python
JSON
CSV
statistics Python Library

## Main menu:

      1) Input/update a month

      2) View Records

      3) Show Statistics

      4) Delete a month 

      5) Export CSV

      6) QUIT

## Running the project

On CMD:

```bash
cd <repository>
python budget.py
```

## Requirements

Python 3.13
